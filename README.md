# fish-farm-roi

Kalkulator sederhana untuk menghitung ROI budidaya ikan secara cepat dan mudah dipahami.

## Struktur proyek

- `index.html` → struktur halaman dan komponen input/output
- `style.css` → styling agar tampilan lebih rapi, mobile-friendly, dan berwarna
- `script.js` → validasi input, perhitungan ROI, dan breakdown biaya

## Fitur utama

- Input mortalitas (`mortality rate`) dan survival rate
- Perhitungan biaya total, omzet, laba/rugi, ROI
- Breakdown komposisi biaya dalam bar persentase
- Metrik tambahan: margin bersih, laba per ekor, BEP harga jual, payback period
- Catatan analisa otomatis berdasarkan hasil simulasi
